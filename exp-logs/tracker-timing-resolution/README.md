### ⚠️ 时间戳注意事项

| 来源 | 时区 | 可靠性 |
|------|------|--------|
| exp-log.docx 中记录的时间 | **UTC+2** (CERN 当地时间) | ✅ 人工记录，可靠 |
| run.log / run_config.yaml | **UTC+8** (本地电脑时区) | ✅ 相对时间差准确 |
| 示波器 .trc 文件绝对时间 | 未知 | ❌ **绝对时间不可用** |
| 示波器 .trc 文件间相对时间差 | — | ✅ 相对时间准确 |

> **关键**: 示波器文件的绝对时间戳完全不准确，不能用于跨系统时间对齐。
> 但同一个 run 内各 .trc 文件的保存时间差是准确的，可用于 spill 结构检测。

## 实验目标

使用 3 层 AC-LGAD strip 探测器组成束流望远镜，测量：
- **时间分辨率** (timing resolution)
- **位置分辨率** (spatial resolution)
